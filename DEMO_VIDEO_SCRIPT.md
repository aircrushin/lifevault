## Production Notes

### Visual Style Guide
- **Color Palette:** Deep black background (#050505), emerald accents (#10b981), gold highlights
- **Animation Style:** Smooth Framer Motion transitions, glass-morphism effects, particle backgrounds
- **Screen Recording:** 60fps, direct interface capture with smooth mouse movements
- **Overlay Graphics:** Thin lines, tech-inspired typography, data visualization elements

### Audio Guidelines
- **Music Style:** Dark electronic, similar to synthwave but more futuristic
- **Voice:** Professional male or female, calm but energetic, slightly compressed for "tech" feel
- **Sound Effects:** Subtle UI clicks, satisfying toggle sounds, data encryption "whoosh"

### Screen Recording Checklist
- [ ] Clean desktop with no distracting apps
- [ ] Browser dev tools closed
- [ ] Consistent 1920x1080 resolution
- [ ] Mock data populated for realistic demo
- [ ] Smooth mouse movements (consider using mouse automation software)
- [ ] Allow animations to complete fully before proceeding

### Alternative Cuts (Optional)
- **30-second teaser:** Hook + Problem + Solution + CTA
- **60-second version:** Cut one feature walkthrough
- **Feature deep-dive (5 min):** Extended screen recordings of each feature with detailed narration
